# Address Book in C

## Objective
To develop a console-based Address Book application for storing and managing contact details efficiently using the C programming language.

## Technologies Used
- C Programming
- Structures
- File Handling
- Pointers
- Functions
- Modular Programming

## Features
- Add new contacts
- View saved contacts
- Search contacts by name or number
- Edit existing contact details
- Delete contacts
- Store data using file handling

## Project Description
This project is a simple Address Book application developed in C that helps users organize and manage contact information. The application performs CRUD operations (Create, Read, Update, Delete) using file handling for permanent data storage.

## Compilation
```bash
gcc *.c
```

## Run
```bash
./a.out
```

## Concepts Used
- Arrays and Structures
- File I/O Operations
- String Handling
- Functions and Modular Design
